title: Import from Google Docs

### Step 1: Open up Your Google Docs

Open up the Google Doc file you'd like to import!

### Step 2: Download Your File as DocX Format

Select the **File** tab of your Google Doc file, head-down to the **Download** and select the docX download.

<Picture
  src="https://static.feynote.com/screenshots/import-instructions/gDocs/gDocs-download-docx-20260126.png"
  alt="Select the download as DocX under the file download tab"
  layout="constrained"
  inferSize={true}
  formats={['avif', 'webp']}
  style="width: 100%; height: auto;"
/>

### Step 3: Locate the Downloaded Files

Open up your system's **File Explorer** and navigate to the folder you've downloaded your Google Doc files too.

### Step 4: Select and Compress Your Files

Select the Docx files you've downloaded, right-click and find the compress to zip option within your native file system.

<Picture
  src="https://static.feynote.com/screenshots/import-instructions/generic/generic-zip-multiple-files-20260126.png"
  alt="Cursor hovering over the compress to zip option in the file explorer"
  layout="constrained"
  inferSize={true}
  formats={['avif', 'webp']}
  style="width: 100%; height: auto;"
/>

### Step 5: Select and Compress Your Files

Finally we can return to Feynote and select the **Browse** button of our File Selection page. Select the zipped file you've created and click **submit**!

<Picture
  src="https://static.feynote.com/screenshots/import-instructions/generic/generic-file-selection-20260126.png"
  alt="Feynote File Section page uploading the zipped file"
  layout="constrained"
  inferSize={true}
  formats={['avif', 'webp']}
  style="width: 100%; height: auto;"
/>

That's it! Your zip file will start to upload and once its done it will autonavigate you back to the import page where we'll convert your notes for you! Please do not close or navigate away from the File Selection page until the zip upload is done.

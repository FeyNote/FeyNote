### Step 1: Locate Your Files

Open your systems **File Explorer** and navigate to the folder containing the files you want to include as part of the transfer.

### Step 2: Select and Compress Your Files

Select the files you want to compress, right-click and find the compress to zip option within your native file system.

### Step 3: Select and Compress Your Files

Finally we can return to Feynote and select the **Browse** button of our File Selection page. Select the zipped file you've created and click **submit**!

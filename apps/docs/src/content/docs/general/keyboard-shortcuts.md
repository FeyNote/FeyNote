---
title: Keyboard Shortcuts
description: All keyboard shortcuts available in FeyNote
---

FeyNote supports keyboard shortcuts for common actions. On macOS, "Mod" refers to the Command key. On Windows and Linux, "Mod" refers to the Ctrl key.

You can customize app-level keyboard shortcuts in **Settings > Keyboard Shortcuts**.

## App-Level Shortcuts

These shortcuts work anywhere in the application and can be customized in Settings. Some shortcuts differ between the desktop app and the browser because certain key combinations are reserved by the browser.

| Action               | Desktop App   | Browser             |
| -------------------- | ------------- | ------------------- |
| Search               | Mod+K         | Mod+K               |
| New Document         | Mod+N         | Mod+Alt+N           |
| New Tab              | Mod+T         | Mod+Alt+T           |
| Close Current Tab    | Mod+W         | Mod+Alt+W           |
| Toggle Left Menu     | Mod+\\        | Mod+\\              |
| Toggle Right Menu    | Mod+Shift+\\  | Mod+Shift+\\        |
| Print                | Mod+P         | Mod+P               |
| Split Right          | Mod+Alt+Right | Mod+Shift+Alt+Right |
| Split Down           | Mod+Alt+Down  | Mod+Shift+Alt+Down  |
| Open Item In New Tab | Mod+Click     | Mod+Click           |

## Text Editing Shortcuts

These shortcuts work when editing a document.

### Formatting

| Action        | Shortcut    |
| ------------- | ----------- |
| Bold          | Mod+B       |
| Italic        | Mod+I       |
| Underline     | Mod+U       |
| Strikethrough | Mod+Shift+X |
| Inline Code   | Mod+E       |
| Insert Link   | Mod+Shift+K |

### Alignment

| Action       | Shortcut    |
| ------------ | ----------- |
| Align Left   | Mod+Shift+L |
| Align Center | Mod+Shift+E |
| Align Right  | Mod+Shift+R |

### Headings

| Action    | Shortcut  |
| --------- | --------- |
| Heading 1 | Mod+Alt+1 |
| Heading 2 | Mod+Alt+2 |
| Heading 3 | Mod+Alt+3 |
| Heading 4 | Mod+Alt+4 |
| Heading 5 | Mod+Alt+5 |
| Heading 6 | Mod+Alt+6 |

### Editing

| Action     | Shortcut    |
| ---------- | ----------- |
| Undo       | Mod+Z       |
| Redo       | Mod+Shift+Z |
| Cut        | Mod+X       |
| Copy       | Mod+C       |
| Select All | Mod+A       |
| Delete     | Del         |

### Lists

| Action       | Shortcut    |
| ------------ | ----------- |
| Bullet List  | Mod+Shift+8 |
| Ordered List | Mod+Shift+7 |
| Task List    | Mod+Shift+9 |

### Block Operations

| Action     | Shortcut    |
| ---------- | ----------- |
| Indent     | Tab         |
| Outdent    | Shift+Tab   |
| Hard Break | Shift+Enter |
